# Keyzone – Tienda de teclados gaming

Proyecto web desarrollado para la asignatura Diseño de Interfaces Web (DIW).

Incluye una landing page, ficha de producto, página de contacto, buscador, carrito funcional, modo claro/oscuro, accesibilidad y varias interacciones además de efectos.
